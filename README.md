# Francesco Azzarello – Portfolio Projects

This repository contains a collection of personal projects developed to improve my skills in **Machine Learning (ML)**, **Data Science (DS)** and **Generative AI**, together with a **technical MVP of a PSD2-compliant instant-payment system (NoC)**.

## Projects Overview

- **Image Captioning Model**  
  End-to-end pipeline to generate image captions using deep learning.

- **Speech Emotion Analyzer**  
  Audio processing + ML classification model to detect emotions.

- **Gradio ML Demo**  
  Simple interface to deploy ML models on the web using Gradio.

- **NoC — Instant Payment MVP**  
  Technical MVP of an instant-payment solution aligned with PSD2/European open banking standards (Node.js + React).  
  _Note: includes only non-sensitive components._

## About this Repository
This repository is intended as a **learning portfolio** and a **technical showcase** for academic applications and future employers.  
Additional projects will be added over time.
